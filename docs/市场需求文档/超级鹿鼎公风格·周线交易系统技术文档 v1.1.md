## 2. 技术指标计算

所有指标基于周线收盘价 `close` 计算。

### 2.1 布林带 (Bollinger Bands)
- **函数**：`ta.bbands(close, length=20, std=2)`（或手动计算）
- **输出列**：`boll_upper`，`boll_mid`，`boll_lower`。
- **带宽**：`bandwidth = boll_upper - boll_lower`。

### 2.2 RSI(14)
- **函数**：`ta.rsi(close, length=14)`，输出0-100序列。

### 2.3 MACD (12,26,9)
- **函数**：`ta.macd(close, fast=12, slow=26, signal=9)`
- **输出列**：`macd_DIF`（DIF），`macd_DEA`（DEA），`macd_hist`（柱值，通常为 `2*(DIF-DEA)`）。

### 2.4 均量线
- `vol_ma5 = volume.rolling(5).mean()`
- `vol_ma20 = volume.rolling(20).mean()`

### 2.5 近4周最高成交量
- `vol_4w_max = volume.rolling(4).max()`

### 2.6 20周均线（趋势过滤，可选）
- `ma20 = close.rolling(20).mean()`

### 3.1 底背离查找算法（V1.2 改用绿柱+两层确认）

不再使用 DIF 线，改用 MACD 绿柱（histogram）。两层确认：
1. **结构性**：最近两个价格波谷，价格更低但绿柱深度更浅（卖方衰竭）
2. **即时性**：当前波谷绿柱已比邻近前一周在缩短（动量已在修复）

```python
from scipy.signal import argrelextrema
import numpy as np

def detect_macd_bullish_divergence(close, hist, order=5):
    if len(close) < order * 2 + 2 or len(hist) < order * 2 + 2:
        return False
    close_vals = close.values.astype(float)
    hist_vals = hist.values.astype(float)
    local_mins = argrelextrema(close_vals, np.less, order=order)[0]
    if len(local_mins) < 2:
        return False
    idx1 = local_mins[-2]
    idx2 = local_mins[-1]
    # 两个波谷绿柱都必须在零轴下方
    if hist_vals[idx1] >= 0 or hist_vals[idx2] >= 0:
        return False
    # 第一层：结构性 — 价格更低，绿柱更浅
    if not (close_vals[idx2] <= close_vals[idx1] and hist_vals[idx2] > hist_vals[idx1]):
        return False
    # 第二层：即时性 — 波谷当周绿柱已比邻近前一周在缩短
    if idx2 == 0 or hist_vals[idx2 - 1] >= 0:
        return False
    return hist_vals[idx2] > hist_vals[idx2 - 1]
```

### 3.2 顶背离查找算法（V1.2 加时效限制）

新增 `max_age` 约束：第二波峰距今超过 `order × 2` 周则背离失效，防止用数月前的旧波峰持续产生卖出信号。

```python
def detect_macd_bearish_divergence(close, dif, order=5):
    local_maxs = argrelextrema(close.values, np.greater, order=order)[0]
    if len(local_maxs) < 2:
        return False
    idx1 = local_maxs[-2]
    idx2 = local_maxs[-1]
    # 第二波峰距今超过 order*2 周则背离已失效
    max_age = order * 2  # 默认10周
    if len(close_vals) - 1 - idx2 > max_age:
        return False
    return (close.iloc[idx2] >= close.iloc[idx1]) and (dif.iloc[idx2] < dif.iloc[idx1])
```

### 4.3 亢奋观察区（卖出）— V1.2 重写

**入口条件**：收盘价 > 上轨，或最高价 > 上轨，或 RSI > 普通超买。
**权限**：极度低估(buy_only)只允许预减仓；价值比>1.2时直接清仓。
**扳机对称化**：首次止盈和加卖共用红柱缩短、顶背离、死叉三个扳机。

```python
def check_sell_signals(row, row_prev, data, current_idx, ext_ob, ob,
                       has_pre_sold, has_first_sold, permission, value_ratio):
    # Zone3入口：收盘价/最高价破上轨，或RSI超买
    enter_zone = (row['close'] > row['boll_upper']) or \
                 (row['high'] > row['boll_upper']) or \
                 (row['rsi'] > ob)
    if not enter_zone:
        return None, 0.0
    
    clear_all = value_ratio > 1.2  # 高估清仓模式
    
    # Step 1: 预减仓 — RSI极端超买（含极度低估）
    if row['rsi'] > ext_ob:
        if clear_all:
            return 'clear_all', 1.0
        return 'pre_sell', 1/3
    
    # 极度低估：预减仓之后禁止常规卖出
    if permission == 'buy_only':
        return None, 0.0
    
    # 三个卖出扳机
    hist_shorter = (row['hist'] > 0) and (row['hist'] < row_prev['hist'])
    bear_div = detect_macd_bearish_divergence(
        data['close'].iloc[:current_idx+1],
        data['dif'].iloc[:current_idx+1])
    death_cross = (row_prev['dif'] > row_prev['dea']) and (row['dif'] < row['dea'])
    
    first_sell_signal = hist_shorter or bear_div or death_cross
    second_sell_signal = hist_shorter or bear_div or death_cross  # 对称
    
    if has_pre_sold or has_first_sold:
        if second_sell_signal:
            if clear_all:
                return 'clear_all', 1.0
            return 'second_sell', 0.5  # 卖出剩余持仓的一半
        return None, 0.0
    else:
        if first_sell_signal:
            if clear_all:
                return 'clear_all', 1.0
            return 'first_sell', 1/3
        return None, 0.0
```

```python
    else:
        return '高估', 0.10, 'sell_only'          # >1.20清仓
```

### 5.2 仓位与交易量计算（V1.2 动态锚定）

**仓位上限锚定**：每周以 `当前持仓总市值 + 现金` 重算总资产，单票资金上限 = 当前总资产 × 20%。

- 每次交易前调用 `position_manager.update_total_capital(current_total_value)` 更新锚定点。
- 单票资金上限 = 总资金 × 单票仓位上限（默认20%）
- 当前持仓占比 = 持有市值 / 单票资金上限
- 本次买入信号对应的理论比例 = 该估值区间仓位上限 × 信号买入百分比
- 实际可买金额 = min(理论比例，剩余可买空间，可用现金)，取整到100股或1手。

**卖出时**：
- 常规卖出数量 = 当前持仓数量 × 信号卖出比例（取整到100股）。
- `clear_all`（高估清仓）时：卖出全部持仓，不取整。

## 6. 回测注意事项

1. **无未来函数**：所有信号仅使用截至当周的数据，`shift(1)` 即上周数据。
2. **时间点**：周五收盘后计算，下周一以开盘价成交。
3. **交易成本**：佣金万三（单边），印花税千0.5（卖出），滑点0.1%。
4. **除权处理**：前复权价格，DCF估值需同步调整内在价值。
5. **信号优先级**：卖 > 买；止盈信号 > 加仓信号；预减仓 > 首次止盈 > 加卖。
6. **参数固定**：回测期间所有可调参数取默认值，不得优化。本章第8节的可选优化仅在实盘评估中采用。

## 9. V1.2 变更汇总（2026-05-14）

基于 V1.1 回测发现的问题进行以下优化：

| # | 变更项 | 旧行为 | 新行为 | 动机 |
|:---|:---|:---|:---|:---|
| 1 | **估值区间** | 4档，无buy_sell | 5档，重新引入buy_sell（0.70-0.80重叠区），档位切分更均匀 | 极度低估独立为免打扰区，为估值再平衡预留空间 |
| 2 | **卖出扳机对称化** | 首次止盈=红柱缩短+顶背离，加卖=仅死叉 | 两者均为红柱缩短+顶背离+死叉 | 死叉不应因"还没卖过"被跳过；加卖不应错失最早的衰竭信号 |
| 3 | **高估清仓** | 无论估值多高都步进卖出 | 价值比>1.2时任一卖出信号直接清仓 | 严重高估不应磨蹭，避免五连卖 |
| 4 | **极度低估预减仓** | buy_only禁止一切卖出 | RSI极端超买时允许预减仓1/3 | 便宜也不能蛮干，极端超买锁利润 |
| 5 | **Zone3入口加最高价** | 仅收盘价>上轨 | 收盘价或最高价>上轨 | 盘中被上轨弹回的上影线需要被捕捉 |
| 6 | **顶背离时效** | 无时限 | 第二波峰逾10周失效 | 数月前的旧波峰不应持续产生卖出信号 |
| 7 | **底背离改用绿柱** | 用DIF线 | 用绿柱+两层确认（结构性+即时性） | 排除DIF假背离，减少滞后 |
| 8 | **买入扳机加绿柱缩短** | 仅金叉+底背离 | 金叉+底背离+绿柱缩短&DIF跌速放缓 | DIF加速下跌时金叉太慢，绿柱缩短+DIF减速更早捕捉拐点 |
| 9 | **仓位动态锚定** | 固定初始资金 | 每周重算为当前总资产 | 市值增长后买入上限同步上调 |
